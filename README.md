# Vesmírná Loď - Space Boat Game

Jednoduchá Android hra, kde řídíte vesmírnou loď napříč celým vesmírem a sbíráte peníze!

## Funkce

- **Ovládání dotykem**: Pohybujte lodí pomocí dotykové obrazovky
- **Sbírání mincí**: Sbírejte padající zlaté mince pro zvýšení skóre
- **Vesmírné pozadí**: Krásné animované hvězdné pole
- **Skóre**: Sledujte, kolik mincí jste sebrali

## Jak hrát

1. Instalujte aplikaci na Android zařízení (min. Android 7.0)
2. Ovládejte vesmírnou loď dotykováním obrazovky
3. Pohybujte se doleva/doprava a sbírejte padající zlaté mince
4. Čím více mincí sesbíráte, tím vyšší skóre!

## Technické požadavky

- Android 7.0 (API 24) nebo vyšší
- Kotlin 1.9.0
- Android SDK 34

## Struktura projektu

- `GameView.kt` - Hlavní herní logika a vykreslování
- `MainActivity.kt` - Vstupní bod aplikace
- Použitý framework: Android SDK + Custom View
